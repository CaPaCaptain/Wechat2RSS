# 私有部署

## 功能特点

- 不限数量订阅
- 智能调整更新频率
- 兼容多种文章格式的全文输出
- 图片和视频代理
- 公众号迁移自动跟随迁移

受微信接口频率限制，订阅数量需要与更新频率平衡，默认策略可以支持400个公众号在24小时内收到更新，详细可参考[更新频率策略设计](https://blog.xlab.app/p/d73537b/)

全文输出效果可参考[公开服务](/list/)
